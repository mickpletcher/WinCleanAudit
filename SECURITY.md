# Security Policy

## Reporting A Security Issue

Please report safety or security issues privately to the maintainers.

Include:

- Clear reproduction steps
- Expected safety behavior
- Actual behavior
- Environment details

## Coordinated Disclosure

Do not publicly share destructive bypass details until maintainers review and
fix the issue.

## Safety Issue Examples

- DryRun performing deletion
- NoPrompt bypassing Execute requirement
- Cleanup logic targeting protected folders
- Cleanup logic modifying startup entries, app inventory, or disk state

## Supported Versions

Version 1.0.0 is currently supported.
